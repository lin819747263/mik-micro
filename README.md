## mik
mik-micro是基于SpringCloudAlibaba快速搭建微服务应用的脚手架

## 项目结构
```aidl
-- mik-cli
    --mik-cache //redis缓冲
    --mik-core  //框架核心
    --mik-db //orm框架
    --mik-log-starter //操作日志模块
    --mik-mvc // web模块
    --mik-swagger-starter //seagger 集成
    --mik-user // 用户和安全模块
```
